## ğŸ› Troubleshooting

### "Error" Message
- **Cause**: Invalid expression (e.g., division by zero)
- **Fix**: Press `C` to clear and start over

### Button Not Responding
- **Cause**: App might be processing
- **Fix**: Wait a moment and try again

### History Not Showing
- **Cause**: No calculations performed yet
- **Fix**: Perform a calculation first (press `=`)

### Scientific Functions Not Working
- **Cause**: Need a result first
- **Fix**: Type a number, press `=`, then use the function
